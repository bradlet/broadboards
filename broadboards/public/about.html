Welcome to BroadBoards!

BroadBoards is a project brought together by three undergraduate students from Portland State University. We took interest in the idea of building a forum-thread based website from the ground up for various reasons and hope to deliver a format that is refreshing compared to the traditional stack frames that are often associated with these sites.

Weâ€™ve now released the first iteration of the work which is focused on creating a stable PERN stack for the site to build off. Future development will expand the user experience by adding features that are common place to social media sites and at some point, begin to branch off in attempt to create something that is unique.



Some future frontend considerations

  • Making the title of threads link to a discussion thread
  • Adding user group options in the sidebar skeleton
  • Implementing a thread editing tool for posting content

Refactoring needs

  • Add additional React components to replace most of the original html
  • Upgrade the existing CSS framework to a different style
  • Convert from CSS3 to either SCSS or SASS

Some future backend considerations

  • Add a many-to-many group for users to post threads on
  • Add queries to sort the thread feed based on predefined preferences
  • Add a profile table for storing user settings
